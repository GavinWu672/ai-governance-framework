# AI Behavior Overrides — Multi-Validator Contract

This domain contract enables four built-in governance validators.
AI agents operating under this contract must:

1. Provide architecture boundary evidence for `feature` and `refactor` tasks.
2. For `kernel-driver` tasks, include IRQL, IOCTL, and cleanup test coverage.
3. Include failure/exception test coverage for all `feature` and `refactor` tasks.
4. For `refactor` tasks, provide regression and interface-stability evidence.
